Functional Attributes

The possible future of the CSS attr rule and how to implement it today with JavaScript.

Cory Simmons - @ccccory

Use cases

  • Grid systems in HTML
  • UI kit elements in HTML

How?

  • Set an attribute in HTML
  • Iterate over elements that have that attribute
  • Perform calculations with JavaScript
  • Set CSS with JavaScript

Why JavaScript?

The attr rule doesn't support a lot of stuff... As in... it only supports setting content so we can't set things like width or background-color. Pretty much useless for our purposes right now.

I'm hopeful for the future of this potentially amazing addition to CSS.

  • https://developer.mozilla.org/en-US/docs/Web/CSS/attr

Wouldn't this be slow?

  • Not really - in my personal experience
  • A few years from now you can remove the JS and it will Just Work™ (well, you'll have to port to CSS)

  • When CSS supports this, you can port your work to CSS and just remove JS so don't worry about performance - especially on smaller sites.
